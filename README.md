# Goforward

Go forward proxy with rate limiting.

# Usage

Point your proxy config at http://localhost:8888

```
.\goforward.exe
```

```
.\goforward.exe -h
Usage of .\goforward.exe:
  -port string
        Proxy listen port (default "8888")
  -proto string
        Proxy protocol (default "http")
```

# Build 

```
go build
```
